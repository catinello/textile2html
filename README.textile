h1. textile2html

---

A simple command line tool to convert textile to html output.

Full credit goes to "txstyle.org":https://txstyle.org/


h3. Requirements

| *Edition* | *Runtime* |
| Script | python 3+ |
| Binary | glibc 2.24+ |

h3. Dependency

"python-textile":https://pypi.python.org/pypi/textile

h3. Compile

bc. $ make depends
$ make compile

h3. Run

h4. Binary

bc. $ dist/txth <input.textile>

h4. Script

bc. $ python txth.py <input.textile>
